### Vim modes
  - Interactive mode (i)
  - Normal mode
  - Visual mode (v)

### Cursor
Move cursor using:
  - left -> h
  - right -> l 
  - up -> k
  - down -> j

### Navigate text
  - words -> w
  - beginning -> b
  - end -> e
  - jump n words -> 3w

### Insert text repeatedly
  -  With editor in normal mode, type: 
     3i + word + esc + enter

### Find next text occurence
  -  With editor in normal mode, type:
     fo -> to find the next o
     3fo -> to find the third occurrence of o
